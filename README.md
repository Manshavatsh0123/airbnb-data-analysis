# Airbnb Data Analysis

A comprehensive data analysis project that explores Airbnb listing data to uncover pricing trends, neighborhood insights, host performance, and customer behavior using Python and SQL.

The objective of this project is to clean, analyze, and visualize Airbnb data to generate meaningful business insights. The project follows a complete data analytics workflow including data preprocessing, exploratory data analysis (EDA), SQL-based analysis, and visualization.

---

## Project Overview

Airbnb hosts millions of property listings across different cities. Understanding pricing patterns, room availability, customer reviews, and host performance can help businesses and property owners make informed decisions.

This project demonstrates how raw Airbnb data can be transformed into actionable insights using Python libraries and SQL queries.

---

## Key Insights

Some of the important findings from the analysis include:

- Entire homes generally have higher prices than private rooms.
- Premium neighborhoods have significantly higher average listing prices.
- A small number of hosts manage a large percentage of listings.
- Highly reviewed properties generally maintain better ratings.
- Popular neighborhoods tend to have lower availability due to higher demand.

## Installation

Clone the repository

```bash
git clone https://github.com/Manshavatsh0123/airbnb-data-analysis.git
```

Move into the project directory

```bash
cd airbnb-data-analysis
```

Install dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Open

```
Airbnb_Data.ipynb
```

Run all notebook cells to reproduce the analysis.
